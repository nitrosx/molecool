A python package for analyzing and visualizing molecules. For the MolSSI Best Practices Workshop on November 7th, 2019 at the University of Pittsburgh, Pittsburgh, PA.  
This repository is currently under development. This can be installed in a development environment by using the following command in the project folder:
`pip install -e .`
